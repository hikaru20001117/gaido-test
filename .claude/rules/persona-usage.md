# ペルソナ壁打ちルール

ユーザーがペルソナでの壁打ちを要求した場合（例: 「鈴木部長のペルソナで壁打ちして」「ペルソナで相談したい」等）、以下の手順で対応すること。

## 特定のペルソナ名が指定された場合

1. Box連携の認証情報（`.box/credentials.json`）を確認する
2. Boxのペルソナフォルダからファイルを検索する:
   ```bash
   python3 tools/box_client.py list --folder-id <base_folder_id>
   ```
3. 該当するペルソナファイルをダウンロードする:
   ```bash
   python3 tools/box_client.py download <ファイルID> --output .claude/skills/personas/<ペルソナ名>.md
   ```
4. ダウンロードしたファイルをReadツールで読み込む
5. ペルソナの役割・思考スタイル・問い返し軸・問い返しのルールに従って、そのペルソナとして振る舞う

## ペルソナ名が指定されていない場合

1. Box連携の認証情報を確認する
2. Boxのペルソナフォルダ内のファイル一覧を取得する
3. AskUserQuestionでペルソナ一覧を提示し、ユーザーに選択させる
4. 以降は上記の手順4-5と同じ

## Box連携が未設定の場合

「Box連携が未設定のため、保存済みペルソナを読み込めません。新しくペルソナを作成する場合は `/gaido-persona-generator` を使ってください」と案内する。

## ペルソナとして振る舞う際のルール

- ペルソナファイルの「問い返しのルール」に従って議論を進める
- 壁打ち中は一貫してそのペルソナとして振る舞う
- ユーザーが壁打ちの終了を示すまで、ペルソナを維持する
