# エラーハンドリング

会話コンテキスト内にエラーメッセージが含まれている場合、以下のルールに従って対応すること。

## 認証エラー（401 authentication_error）

会話コンテキストに `authentication_error`、`Failed to authenticate`、`Invalid authentication credentials` のいずれかが含まれる場合、以下のガイダンスをユーザーに表示してから通常のフロー（welcome-message等）に復帰すること。

### ガイダンス内容

1. **何が起きているか**: Claudeの認証情報が無効になっているため、APIへの接続でエラーが発生した
2. **原因**: Claudeのプラン変更（Standard↔Premium等）やトークンの期限切れにより、認証トークンが無効化されることがある
3. **対処法**: 別のブラウザタブで https://claude.ai からログアウトし、再度ログインしてから、GAiDoを再起動する
4. **即座の回避策**: エラーが表示されても、そのままチャットを続けると正常に動作する場合がある。まずはこのまま続けてみることを推奨する
