# エゴグラム変換マトリクス

## 概要

エゴグラムの5値（CP/NP/A/FC/AC）を性格特性リスト（OK/NOT-OK）に変換するためのマトリクス。

※ 50問の質問セットは本バージョンに含まない。エゴグラム診断は外部ツールで事前に実施し、5値を直接入力すること。

## 変換ルール

診断で算出された点数に応じて、参照する特性が変わる:

| 点数 | 参照箇所 |
|------|---------|
| 0〜13点 | 左側（低）のOK/NOT-OK |
| 14〜16点 | 左右両方のOK/NOT-OK |
| 17〜30点 | 右側（高）のOK/NOT-OK |

## 特性テーブル

### CP（批判的親: Critical Parent）

| | 低（0-13） | 高（17-30） |
|--|-----------|------------|
| **OK** | ルールやしきたりにとらわれない / 変化に適応する / 調整が利く / 非難・攻撃しない / 先入観をもたない / 情緒が安定している | ルールを遵守する / 古くからのしきたりや風習を大切にする / 義理堅い / 評価を下す / 生真面目 / 倫理的、人道的 |
| **NOT-OK** | 物事を粘り強く最後までやりきることが苦手 / 片付け整えることが苦手 / 規範を軽視する / 節度がない / 倫理的でない / 義理堅さに欠ける | 物事にシビアすぎる / 自分の価値観を譲らない / 曖昧なことが嫌い / 思い込みが激しい / 物事を批判的に見る / 高圧的 / 偏見が強い |

| **NOT-OK** | 同情しない / 気配りが足りない / 気を回さない / 相手を考慮しない / 優しくない / 労い・思いやりが足りない / 相手のことを考えない | 出しゃばりすぎる / かまい過ぎる / 相手を保護しすぎる / 相手に干渉しすぎる / 奉仕しすぎる / 溺愛する / 主体性を阻害する |

## エゴグラム特性の解釈テンプレート

5値のうち、値が高い順に以下の形式で解釈を記述する:

```
- **{要素}={値}（最高）**: {OK特性から主要なものを2-3個抜粋して自然な文章にする}
- **{要素}={値}（高）**: {同上}
- **{要素}={値}（中）**: {左右両方のOK特性から特徴的なものを抜粋}
- **{要素}={値}（低）**: {OK特性から主要なものを2-3個抜粋}
- **{要素}={値}（最低）**: {同上}
```

例（honbucho.md: CP=23/NP=6/A=27/FC=17/AC=8）:
```
- **A=27（最高）**: 事実・データ・論理で判断する。感情より根拠を重視
- **CP=23（高）**: 基準が高く批判的。「なぜ?」を繰り返し、曖昧さを許容しない
- **FC=17（中）**: 筋の通った案件・説明には乗り気になる。面白いと思ったら前のめりになる
- **NP=6（低）**: フォローも褒めもしない。ドライ
- **AC=8（低）**: 忖度ゼロ。空気を読んで質問を緩めることはない
```
